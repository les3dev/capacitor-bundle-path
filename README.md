# @les3dev/capacitor-bundle-path

Retrieve the file system path to a bundled asset, or undefined if the asset is not found

## Install

```bash
cd ..
git clone @les3dev/capacitor-bundle-path
cd myproject
pnpm add ../capacitor-bundle-path
pnpm i
pnpx cap sync
```
